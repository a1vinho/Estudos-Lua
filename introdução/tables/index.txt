- Na linguagem de programação lua,temos uma única estrutura de dados em toda a linguagem,essa estrutura de dados é chamada de tabela (table).As tabelas são utilizadas para a crianção de arrays simples,objetos complexos e dicionarios.Com uma única estrutura podemos criar vários tipos.Com as tabelas podemos criar arrays indexados por numeros e strings,execto nil.Diferente de algumas estrutura especificas as tabelas não tem tamanho fixo e podem crescer de acordo com nossa necessidade.

- As tabelas armazenam pares chave e valor,onde para ambos podemos adicionar qualquer tipo de dados,menos nil.Tudo em lua de arrays simples a objetos complexos funciona atráves de table.Quando os indices são numericos lua percebe aquela estrutura como um array e passa a otimiza-la como um array interno da propria linguagem

* criando uma tabela simples {
    - local langs = {"javascript","python","php","lua"} -- Esse tipo de tabela passa a ser visto como um array pela linguagem lua,isso por que seus indices são numeros consecutivos. {
        - para acessamos dados do array langs utilizamos os indices 
        print(langs[1])
    }

    * interando sobre arrays consecutivos

    for index,value in ipairs(langs) do 
        print(value)
    end; - utilizamos a função ipairs que é totalmente feita para arrays 

    * interando sobre objetos ou dicionarios 

    - local user = {
        name = "alvaro",
        idade = 19
    } -- Essa tabela é um objeto / dicionario 
    for (key,value in pairs(user)) do 
        print(user[key])
    end; - o pairs é uma função geralmente utilizada em objetos ou dicionarios,ela não garante ordem nos dados,diferente da função ipairs.Leia em detalhes o arquivo da pasta controle de fluxo
}

* principais funções dentro de uma table {
    1 - table.insert(table,<pos>,value): Essa função é utilizada para adicionar um novo elemento dentro de uma tabela,table é a tabela referenciada,<pos> é a posição do novo elemento,caso não informado o novo elemento irá para o fim da tabela e value é o elemento a ser adicionado dentro da tabela. {
        local nums = {0,1,2,3};

        table.insert(nums,4);

        print(nums)
    }
    2 - table.remove(table,<pos>): Para removemos um elemento especifico dentro de uma tabela,utilizamos a função table.remove. o parametrô table é a tabela onde contém o elemento que desejamos remover,já o pos é a posição do elemento que desejamos remover dentro da tabela,caso não seja informado,a função irá remover o ultimo elemento: {
        local nums = {0,1,2,3};
        table.remove(nums,2);

        -- {0,2,3}
    }
    3 - table.move(origem, inicio, fim, destino, destino_inicio): Essa função é responsável por copia de maneira eficiente e perfomatica os elementos de uma tabela de origem para uma tabela de destino,ela também pode ser utilizada para copia elemento para a mesma tabela de origem. {
        - Vamos destricha os parametros um por um 

        1 - origem: A tabela de onde os valores serão copiados 
        2 - inicio: Indice inicial de onde os valores irão começar a ser copiados.
        3 - fim: Indice final até onde a copia será feita.
        4 - destino: Tabela de destino onde os valores copiado serão armazenados.
        5 - destino_inicio: Indice inicial da tabela de destino,onde os valores copiados começarão a ser inseridos. 


        - exemplo: {
            local origem = { 10, 20, 30, 40, 50 }
            local destino = {}

            table.move(origem, 1, 5, 1,destino);


            for i,v in ipairs(destino) do
                print(v)
            end

            * Nesse exemplo estamos copiados os valores das posições 1 e 5 para a tabela de destino 
        }
         - exemplo: {
            local origem = { 10, 20, 30, 40, 50 }
            local destino = {}

            table.move(origem, 1, 5, 4);


            for i,v in ipairs(origem) do
                print(v)
            end
            * Nesse código estamos utilizando o table.move para sobreescreve elementos dentro da tabela de origem,podemos utilizar o table.move para sobreescreve elementos já existentes dentro da tabela
         }
    }
    4 - table.pack(...): Essa função é responsável por criar uma nova tabela,baseado em uma lista de elementos dentro da mesma: {
        local langs = {"javascript","php","python","csharp"};

        local newTable = table.pack(langs);

        - Além do mais essa função guarda uma propriedade n,essa propriedade contém a quantidade de elementos dentro da nova tabela 

        print(newTable.n)
        - table.pack trabalha de maneira eficiente com valores nil em tabelas com valores consecutivos,essa função preserva os valores nil dentro da tabela,tornando a eficiente para trabalha com valores nulos,além do mais o table.pack preserva o tamanho real da tabela,pela propriedade n.
        - o table.pack é principalmente utilizando quando não sabemos os dados que a lista poderá armazena,essa função previne comportamento inesperados.
    }
    5 - table.sort(table,[comp]): Essa função é utilizada para organiza uma tabela,por padrão essa função organiza uma tabela em ordem ascedente (crescente - numerica ou alfabetica).Podemos também alterar o comportamento padrão atráves do parametro comp,esse parametro é uma função onde podemos personaliza a ordenação: {

        - Ordenação padrão

        local frutas = {"banana", "laranja", "abacaxi"}
        table.sort(frutas)
        -- Resultado: {"abacaxi", "banana", "laranja"}
        - a função table.sort so funciona em array com valores ou indices numericos consecutivos. 1,2,3,4.
        - a função sort ela alterar a tabela original e não retorna uma nova.
    }
    6 - table.unpack(list,inicio,fim = #list): Essa função é responsável por desempacotar uma tabela,transformando-a em valores vareados que podem ser amarzenados em variaveis.Lembrando que essa função so funciona em tabelas com valores e indices consecutivos.Essa função pega os valores de uma tabela e os retorna como valores separados {
        - Essa função é utilizada quando precisamos desestrutura um array consecutivo 
        local nums = {1,2,3,4,5,6,50,200,100};
        local max = math.max(table.unpack(nums)) // 100

        * explicando cada argumento:
        table: a tabela que você quer “desempacotar”.

        i: índice inicial (opcional, padrão é 1).

        j: índice final (opcional, padrão é #table, ou seja, o tamanho da tabela).

        - em resumo essa função é o oposto da função table.pack que pega os valores separados e armazena em uma tabela.

        local t = {10, 20, 30, 40, 50}

        print(table.unpack(t, 2, 4))  -- pega do índice 2 ao 4
    }
    7 - table.concat(list,sep,inicio,fim):Essa função é responsável por junta uma tabela passada como argumento utilizando um seperador especifico.Em outras palavras, essa função junta uma tabela em uma única string,utilizando um separador especifico.Lembrando mais uma vez que essa função funciona apenas em tabelas com valores ou indices consecutivos {
        local nums = {1,2,3,4,5}
        print(table.concat(nums)) 12345

        - Essa função transforma toda a tabela em uma única string.

        local text = {"Lua","é","legal"};
        print(table.concat(text,'-> ')) Lua->é->legal

        * podemos também utiliza os intervalos 

        local letras = {'a','b','c','d','e'};
        print(table.concat(letras,':',3,5)) // c:d:e
    }
}

- Lembrando que tabelas também pode ser arquivos ou modulos.Internamente as tabelas são diferentes ou seja hibridas.
